Desktop launcher for Windows

바탕화면에서 바로 여는
즐겨찾기 실행 위젯

Favorite Widget은 Windows 10/11 데스크톱에서 앱, 파일, 링크를 한 번에 정리하고 바로 실행할 수 있게 만든 즐겨찾기 위젯입니다. 빠른 실행은 유지하고, 설정은 복잡하지 않게 정리했습니다.

빠른 실행 + 간단한 관리

바탕화면 위젯과 설정 창을 분리해서, 실행은 빠르게 하고 관리는 차분하게 정리했습니다.

01

데스크톱 실행 동선 최소화

자주 여는 프로그램과 파일을 바탕화면 위젯에서 바로 실행합니다.

02

실행 파일 기본 아이콘 유지

카카오톡 같은 앱은 exe 기본 아이콘을 그대로 가져와서 인지성이 좋습니다.

03

GitHub Pages용 정적 소개 페이지 포함

설치 파일, 스크린샷, 기능 설명을 바로 공개할 수 있는 `docs/` 구조를 같이 제공합니다.

How it works

복잡한 런처 대신, 필요한 것만 남겼습니다

1. 즐겨찾기 등록

앱(.exe), 문서, 링크를 설정 창에서 추가합니다.

2. 아이콘과 이름으로 확인

메인 위젯과 목록에서 실행 대상이 바로 구분됩니다.

3. 바탕화면에서 즉시 실행

앱을 탐색기나 시작 메뉴로 다시 찾지 않아도 됩니다.

First use guide

처음 사용할 때는 이렇게 보면 됩니다

이 페이지에서 가장 헷갈리기 쉬운 부분은 설정 창과 메인 위젯의 역할입니다. 설정 창은 즐겨찾기를 등록하고 정리하는 화면이고, 메인 위젯은 등록한 항목을 바로 실행하는 화면입니다.

STEP 1 설정 창에서 등록

앱 이름과 실행 파일 경로를 저장합니다.

STEP 2 메인 위젯에서 실행

바탕화면 위젯에서 아이콘을 눌러 바로 엽니다.

FAQ

자주 묻는 질문

Favorite Widget에는 무엇을 등록할 수 있나요?

실행 파일(.exe), 문서/파일, 그리고 `http` 또는 `https` 링크를 등록할 수 있습니다.

설정은 어디에 저장되나요?

`%LOCALAPPDATA%\FavoriteWidget\favorite-widget.ini` 파일에 저장됩니다.

설치 없이도 실행할 수 있나요?

가능합니다. `build/favorite_widget.exe`를 직접 실행해도 되고, 인스톨러(`FavoriteWidget-0.2.0-win64.exe`)를 사용해도 됩니다.
