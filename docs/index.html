Favorite Widget

Windows shortcut launcher

Launch apps, folders, and URLs in one compact widget.

Favorite Widget keeps the existing INI format and adds a separated WinUI 3 settings app. You can still use an existing `favorite-widget.ini` file and switch to the modern settings window anytime.

  • Windows 10 / 11
  • Open apps, folders, and URL shortcuts
  • Add, edit, and remove favorites
  • WinUI 3 settings app with instant file-sync

One-window workflow

Keep widget usage lightweight and launch a full settings tool only when needed.
