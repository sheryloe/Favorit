빠른 실행 / INI 호환

앱·폴더·URL을 가장 빠르게 실행.

Favorite Widget은 게임 위나 작업 화면 옆에서 바로 꺼내 쓰는 경량 실행 위젯입니다. 기존 favorite-widget.ini를 그대로 읽고, 설정은 분리된 WinUI 3 창에서 정리합니다.

다운로드 GitHub
  • Windows 10 / 11
  • INI 경로 그대로
실제 프로그램 화면이 먼저 보이도록 정리했습니다.

메인 위젯은 전면, 설정 화면은 보조 카드로 배치해 첫 화면 인지성을 높였습니다.

구성: Win32 위젯은 유지하고, 설정만 WinUI 3로 분리합니다.

즉시

한 화면 실행

자주 쓰는 앱과 폴더를 작은 위젯 한 장에 두고 바로 실행합니다.

호환

기존 데이터 유지

INI 포맷과 경로를 그대로 유지해 마이그레이션 비용을 없앴습니다.

분리

설정 UI 정돈

WinUI 3 설정 창에서 목록, 폼, 명령 버튼을 분리해 관리 흐름을 단순하게 만듭니다.

실행 흐름

등록과 실행을 짧게 유지하는 기본 동선입니다.

01 추가

앱, 폴더, URL을 등록하면 위젯에 바로 반영됩니다.

02 정리

설정 창에서 라벨과 경로를 수정하고 순서를 정돈합니다.

1클릭 우선

메인 위젯은 실행 버튼 밀도와 가독성에만 집중합니다.

즉시 반영

INI 변경 내용은 위젯에 다시 읽혀 현재 상태와 어긋나지 않습니다.

기존 기반 유지

핵심 실행부는 Win32로 유지하고, 설정 경험만 분리해 현대화합니다.

스크린샷

메인 위젯을 중심으로 보고, 설정 화면은 보조로 확인하는 구성을 사용합니다.

페이지 연출 기준

위젯 자동 갱신

FAQ

자주 묻는 내용을 짧고 명확하게 정리했습니다.

설정 앱이 없어도 위젯은 동작하나요?

네. 위젯은 INI만 읽을 수 있으면 정상 동작합니다.

기존 데이터는 그대로 유지되나요?

네. INI 포맷과 경로는 변경하지 않습니다.
