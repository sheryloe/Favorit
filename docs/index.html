Favorite Widget

Windows Favorite Launcher

바탕화면에서 바로 여는
윈도우 위젯형 즐겨찾기 런처

Favorite Widget은 자주 쓰는 실행 파일, 문서, URL을 Windows 바탕화면에서 위젯처럼 정리해 두고 바로 실행할 수 있게 만든 데스크톱 유틸리티입니다. 목표는 복잡한 런처가 아니라 “손이 가는 자리”에 두는 빠른 실행 경험입니다.

  • Windows 10 / 11
  • 파일, 폴더, URL 등록
  • 설정 창에서 추가 / 수정 / 삭제
  • 설치 파일과 공개 페이지 동시 제공

왜 지금 더 좋아질 수 있나

현재 구조만으로도 충분히 쓸 만하지만, 디자인을 좀 더 Windows 위젯답게 밀어 올리면 서비스 완성도가 확 올라갑니다.

C++ / Win32 API / CMake / NSIS

Current build: 0.2.0

Main Widget

자주 쓰는 실행 대상이 눈에 띄는 위치에 배치됩니다.

Settings

추가, 수정, 삭제가 별도 설정 창에서 이루어집니다.

Release Notes

지금 좋은 점

  • 설치 파일이 이미 준비되어 있습니다.
  • 스크린샷 자산과 Pages 설명 페이지가 있습니다.
  • 위젯형 콘셉트가 분명해 개선 방향이 뚜렷합니다.
Service TODO

디자인 리뉴얼

실제 Windows 위젯처럼 보이는 패딩, 그림자, 밀도, 아이콘 시스템을 더 강하게 가져갑니다.

사용성 강화

드래그 정렬, 검색, 그룹화, 전역 단축키가 들어가면 완성도가 크게 올라갑니다.

배포 품질

설정 백업/복원, 시작 프로그램 등록, 설치 파일 서명까지 가야 서비스 제품처럼 보입니다.

Windows 데스크톱에서 자주 쓰는 것을 더 빨리 여는 경험에 집중한 프로젝트입니다.
